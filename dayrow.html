OVERVIEW

THE CLIENT

Day Row Coffee is a conceptual cafe brand based in Seoul that focuses on a refined and intentional coffee experience, emphasizing simplicity and atmosphere.

THE MISSION

The goal of this project was to redesign the visual branding and develop a responsive website that reflects the updated identity. The focus was on creating a clear, cohesive interface with intuitive navigation and subtle interactions.

Below are my contributions to the research, design, and web development.

IDENTIFYING THE PROBLEM

Day Row Coffee's existing identity lacked a clear visual direction and did not effectively communicate the brand's intended atmosphere.

Through evaluation of the original concept, key issues were identified that impacted both branding clarity and user experience. These insights guided the redesign process.

INCONSISTENT BRANDING

Visual elements such as typography, color, and layout were not cohesive, making the brand feel unclear and unfocused.

WEAK DIGITAL PRESENCE

The interface did not reflect the intended atmosphere of the cafe, reducing its ability to create a strong first impression.

LIMITED USER ENGAGEMENT

The lack of interaction and structure made it difficult for users to explore content and connect with the brand.

OUR APPROACH

Translate Day Row Coffee's identity into a clear visual system and intuitive digital experience.

DESIGN EXECUTION

The interface embraces a minimalist design, using clean layouts and deliberate spacing to create clarity and focus. A clear visual hierarchy helps guide users through the menu with ease, making key information easy to scan while keeping the overall experience calm and intentional.

VISUAL DIRECTION

This simplicity is balanced with illustrative elements and moments of creativity throughout the menu, adding warmth, personality, and a sense of excitement as users explore. Together, these choices translate Day Row Coffee's identity into a digital experience that feels cohesive and engaging.

REFLECTION

COLLABORATION

Working collaboratively through GitHub was a key part of this project. Managing branches, merging changes, and resolving conflicts helped me better understand version control workflows.

KEY TAKEAWAYS

Accidental overlapping edits and inconsistencies between files created challenges when merging our work, especially across layout and styling. These obstacles highlighted the importance of clear communication, consistent structure, and shared design decisions in early development stages. Overall, the experience strengthened both my technical skills and my ability to collaborate effectively within a team environment.
