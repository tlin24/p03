OVERVIEW

THE CLIENT

Day Row Coffee is a conceptual café brand based in Seoul that focuses on a refined and intentional coffee experience, emphasizing simplicity and atmosphere.

THE MISSION

The goal of this project was to redesign the visual branding and develop a responsive website that reflects the updated identity. The focus was on creating a clear, cohesive interface with intuitive navigation and subtle interactions.

Below are my contributions to the research, design, and web development.

IDENTIFYING THE PROBLEM

Day Row Coffee's existing identity lacked a clear visual direction and did not effectively communicate the brand's intended atmosphere.

Through evaluation of the original concept, key issues were identified that impacted both branding clarity and user experience. These insights guided the redesign process.

INCONSISTENT BRANDING

Visual elements such as typography, color, and layout were not cohesive, making the brand feel unclear and unfocused.

WEAK DIGITAL PRESENCE

The interface did not reflect the intended atmosphere of the café, reducing its ability to create a strong first impression.

LIMITED USER ENGAGEMENT

The lack of interaction and structure made it difficult for users to explore content and connect with the brand.

OUR APPROACH

Translate Day Row Coffee's identity into a clear visual system and intuitive digital experience.
