INTERACTIVE ELEMENTS

In Text Link

linkedin: @terrylin.linkedin

HTML

<section class="contact-cont"> <p> linkedin: <a href="https://www.linkedin.com/in/terry-lin-54bb851bb/" target="_blank"> @terrylin.linkedin </a> </p> </section>

CSS

.contact-cont p { margin: 0 0 1rem 0; font-family: "Simplon Mono", monospace; font-size: 1rem; line-height: 1.6; } .contact-cont a { color: #2323ff; text-decoration: none; }

Navigation Link

  • HOME
  • PROJECTS
  • CONTACT

HTML

<nav class="navbar"> <ul> <li><a href="#">HOME</a></li> <li><a href="#">PROJECTS</a></li> <li><a href="#">CONTACT</a></li> </ul> </nav>

CSS

.navbar { display: flex; align-items: center; justify-content: center; padding: 0.5rem 10%; } .navbar ul li { list-style-type: none; display: inline-block; padding: 10px 20px; } .navbar ul li a { position: relative; color: #000000; font-family: "NeueMontreal", sans-serif; font-weight: bold; text-decoration: none; font-size: 1rem; letter-spacing: 0.1em; } .navbar ul li a::after { content: ""; position: absolute; left: 0; bottom: -4px; width: 100%; height: 2px; background-color: #2323ff; transform: scaleX(0); transform-origin: left; transition: transform 0.5s ease; } .navbar ul li a:hover { color: #2323ff; } .navbar ul li a:hover::after { transform: scaleX(1); }

Button

HTML

<button class="btn">VIEW PROJECT</button>

CSS

.btn { font-family: 'Simplon Mono', sans-serif; padding: 5px 20px; border: 2px solid #2323ff; background: none; color: #2323ff; border-radius: 6px; cursor: pointer; transition: transform 0.3s ease, background-color 0.3s ease; background-color: #F5F5F5; } .btn:hover { transform: scale(1.05); background-color: #f2f0ef; }

Text Input Field with Label

HTML

<div class="form-box"> <label for="name">NAME:</label> <input type="text" id="name" placeholder="enter your name" /> <label for="email">EMAIL:</label> <input type="text" id="email" placeholder="enter your email" /> </div>

CSS

.form-box { display: flex; flex-direction: column; gap: 1.2rem; max-width: 420px; } .form-box label { font-family: "Simplon Mono", monospace; font-size: 0.75rem; letter-spacing: 0.08em; color: #2323ff; } .form-box input { width: 100%; padding: 0.8rem 0; font-size: 1rem; font-family: "NeueMontreal", sans-serif; background: transparent; border: none; border-bottom: 1px solid #000000; transition: border-color 0.3s ease; outline: none; } .form-box input:focus { border-bottom: 1px solid #2323ff; } .form-box input::placeholder { color: #898989 }

Radio Button with Label

HTML

<div class="radio-box"> <label> <input type="radio" name="role" checked /> <span>Developer</span> </label> <label> <input type="radio" name="role" /> <span>Designer</span> </label> </div>
