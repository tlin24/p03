INTERACTIVE ELEMENTS

In Text Link

Visit @SFU for more information.

HTML

<p>Visit <a class="tlink" href="https://www.sfu.ca/">@SFU</a> for more information.</p>

CSS

.tlink { font-family: "Simplon Mono", sans-serif; text-decoration: none; color: #2323ff; } .tlink:hover { color: #000000; text-decoration: underline; }

Navigation Link

  • HOME
  • PROJECTS
  • CONTACT

HTML

<nav class="navbar"> <ul> <li><a href="#">HOME</a></li> <li><a href="#">PROJECTS</a></li> <li><a href="#">CONTACT</a></li> </ul> </nav>

CSS

.navbar { display: flex; align-items: center; justify-content: center; padding: 0.5rem 10%; } .navbar ul li { list-style-type: none; display: inline-block; padding: 10px 20px; }

Button

HTML

<button class="portfolio-btn">VIEW PROJECT</button>

CSS

.portfolio-btn { padding: 9px 20px; border-radius: 50px; cursor: pointer; border: 0; background-color: #f5f5f5; letter-spacing: 1.5px; font-size: 15px; transition: all 0.5s ease; } .portfolio-btn:hover { letter-spacing: 3px; background-color: #2323ff; color: #ffffff; }

Text Input Field with Label

HTML

<label for="name">NAME</label> <input type="text" id="name" placeholder="Enter your name" />

CSS

label { font-family: "NeueMontreal", sans-serif; font-weight: bold; } input { width: 100%; background: transparent; border: 0; border-bottom: 2px solid #000000; outline: none; }

Radio Button with Label

HTML

<label class="radio-option"> <input type="radio" name="role" checked /> <span>Designer</span> </label>

CSS

.radio-option input[type="radio"] { display: none; } .radio-option span { display: inline-flex; align-items: center; padding: 10px 20px 10px 10px; border-radius: 25px; }

Image as a Link

HTML

<a href="#" class="image-link"> <img src="img/dayrow.png" alt="Project preview" /> </a>

CSS

.image-link img { max-width: 420px; border-radius: 18px; display: block; }
